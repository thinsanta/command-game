How to play this game?

Start it with the command "node app.js".
Give it width and height (example 5 5) and press ENTER.
Give it position (example 0 2) and press ENTER.
Give ONE command at a time (example 1) and press ENTER.
Give 0 to see the result.

Exmaple of a game:

Enter table size (width height): 5 5
Enter object's starting position (x y): 0 2
Enter command (0 to quit): 3
Enter command (0 to quit): 1
Enter command (0 to quit): 1
Enter command (0 to quit): 3
Enter command (0 to quit): 2
Enter command (0 to quit): 0

result [ 2, 1 ]
